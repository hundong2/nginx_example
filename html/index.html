🚀 완전한 Nginx 가이드

초보자에서 전문가까지 단계별 학습

📖 전체 가이드 문서 🐙 GitHub 저장소 ⚡ 빠른 시작
레벨 1

기초 (Beginner)

nginx의 기본 개념과 설정을 배웁니다

  • 🌐 기본 웹 서버 설정 실행 가능
  • 🏠 Virtual Hosts (가상 호스트) 실행 가능
  • 🔒 기본 보안 헤더 실행 가능
레벨 2

중급 (Intermediate)

프록시, 로드 밸런싱, SSL을 배웁니다

  • 🔄 리버스 프록시 준비 중
  • ⚖️ 로드 밸런싱 준비 중
  • 🔐 SSL/TLS 설정 준비 중
  • 💨 캐싱 및 압축 준비 중
레벨 3

고급 (Advanced)

고급 기능과 보안을 배웁니다

  • 🚦 Rate Limiting 준비 중
  • 🗂️ 고급 캐싱 준비 중
  • 🛡️ 고급 보안 준비 중
  • 🔍 로그 분석 및 모니터링 준비 중
레벨 4

전문가 (Expert)

최고 수준의 nginx 활용법을 배웁니다

  • 🚀 성능 최적화 준비 중
  • 🌙 Lua 스크립팅 준비 중
  • 📊 고급 모니터링 준비 중
  • 🔄 블루-그린 배포 준비 중

⚡ 빠른 시작

모든 예제 실행
docker compose --profile all up -d
기본 서버만 실행
docker compose --profile basic up -d
보안 헤더만 실행
docker compose --profile security up -d
모든 서비스 중지
docker compose down
