Virtual Hosts 예제 - 첫 번째 사이트

이 페이지는 site1.local 도메인으로 접근할 수 있습니다.

🎯 브랜딩

각 사이트는 독립적인 브랜딩과 디자인을 가질 수 있습니다.

📊 로깅

사이트별로 별도의 액세스 로그와 에러 로그를 관리합니다.

⚙️ 설정

각 가상 호스트마다 다른 nginx 설정을 적용할 수 있습니다.

🔒 보안

사이트별로 다른 보안 정책과 SSL 인증서를 사용할 수 있습니다.

테스트 방법:

  • 브라우저: http://site1.local:8002
  • curl: curl -H "Host: site1.local" http://localhost:8002
