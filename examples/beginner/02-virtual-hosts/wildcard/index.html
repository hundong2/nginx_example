🌟 와일드카드 서브도메인

Wildcard Subdomain - *.example.local

와일드카드 서버 블록 예제

이 페이지는 *.example.local 패턴에 매칭되는 모든 서브도메인으로 접근할 수 있습니다.

🎯 nginx 설정

server { listen 80; server_name ~^(?<subdomain>.+)\.example\.local$; # 서브도메인 캡처 변수 사용 add_header X-Subdomain $subdomain; root /var/www/wildcard; index index.html; }

테스트 방법:

다음과 같은 방법으로 테스트할 수 있습니다:

# hosts 파일에 추가 (테스트용) 127.0.0.1 api.example.local 127.0.0.1 blog.example.local 127.0.0.1 shop.example.local # curl로 테스트 curl -H "Host: api.example.local" http://localhost:8002 curl -H "Host: blog.example.local" http://localhost:8002 curl -H "Host: shop.example.local" http://localhost:8002

활용 예시:

  • api.example.local - API 서버
  • blog.example.local - 블로그
  • shop.example.local - 온라인 쇼핑몰
  • admin.example.local - 관리자 패널

💡 정규표현식 설명

~^(?<subdomain>.+)\.example\.local$

  • ~: 정규표현식 매칭
  • ^: 문자열 시작
  • (?<subdomain>.+): 서브도메인을 캡처하여 $subdomain 변수에 저장
  • \.example\.local: 고정된 도메인 부분
  • $: 문자열 끝
