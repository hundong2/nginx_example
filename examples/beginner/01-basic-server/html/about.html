nginx 기본 서버에 대해

🏗️ 현재 설정

  • 포트: 8001 (Docker 포트 매핑)
  • 문서 루트: /usr/share/nginx/html
  • 설정 파일: nginx.conf
  • 로그 위치: /var/log/nginx/

📝 nginx.conf 주요 설정

server {
    listen 80;
    server_name localhost;
    root /usr/share/nginx/html;
    index index.html index.htm;
    
    location / {
        try_files $uri $uri/ =404;
    }
}

🔧 Docker Compose 설정

  • nginx:alpine 이미지 사용
  • 포트 8001:80 매핑
  • 설정 파일과 HTML 볼륨 마운트
  • 헬스체크 설정
